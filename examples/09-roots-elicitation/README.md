# 09 — Roots и Elicitation

Две оставшиеся client-фичи:

- **Roots** — сервер спрашивает клиента «какие директории/URI ты видишь?» (`roots/list`), клиент шлёт `notifications/roots/list_changed` при изменении workspace.
- **Elicitation** — сервер просит у пользователя структурированный ввод в середине операции (`elicitation/create`) с JSON Schema для формы.
