# 12 — Security

Три класса атак, которые специфичны именно для MCP, плюс демо каждой на собственном сервере:

- **Prompt injection через tool description** — модель читает описание tool как часть контекста, злой сервер может внедрить инструкции.
- **Tool shadowing** — один сервер регистрирует tool с тем же именем, что у другого, и перехватывает вызовы.
- **Confused deputy** — MCP-сервер имеет больше привилегий, чем пользователь, и LLM можно уговорить сделать то, на что сам пользователь не имел бы права.

Плюс базовая гигиена: валидация входов на границе, логирование в stderr (не stdout!), ограничение файловых roots.
